## How to use

1. **New decision needed:** add a row to "Open" with a number, the question, who's deciding, and a target date.
2. **Decision made:** write an ADR, link it from this file under "Recently closed," and remove the row from "Open."
3. **Quarterly:** walk this file. Stale rows ("decide by" passed) get re-prioritized or explicitly killed.
